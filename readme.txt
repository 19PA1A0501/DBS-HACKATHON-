This is a Django project (backend) for Payments
payment is an App in Django and MYSQL database is used to store the details of payments and cards

to run this django app install django and then in this directory, run this command python manage.py runserver 
then Go to http://127.0.0.1:8000/payment/ for payment frontend
Go to http://127.0.0.1:8000/admin/ (username: admin, password: test) for backend SQL Table
